Pandas Practice — Beginner to Advanced
A hands-on practice repository for learning and revising Python Pandas, built while brushing up on data analysis skills alongside SQL.
📌 About
This repo contains structured practice questions and my own solutions covering Pandas fundamentals — from creating Series and DataFrames to filtering, handling missing data, groupby operations, and file I/O. Each question is solved, tested, and corrected for common syntax mistakes (a great reference for anyone else relearning Pandas too).
📂 Contents
pandas_beginner_questions.pdf — Full set of Beginner Level practice questions with a sample dataset
21-07_python.py — My worked solutions and practice code
🧠 Topics Covered (Beginner Level)
Series creation & attributes (dtype, size, index)
DataFrame basics (info(), describe(), shape, dtypes)
Viewing & selecting data ([], .loc[], .iloc[], .at[])
Basic operations (sort_values(), sort_index(), drop(), rename(), astype())
Filtering data (isin(), .between(), .query())
Handling missing data (isna(), dropna(), fillna(), interpolate())
GroupBy basics (groupby(), agg(), count(), mean())
Apply & Map (.apply(), .map(), lambda functions)
Input/Output (to_csv(), read_csv(), to_dict())
